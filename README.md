This project is created as a visualization tool and an implementation of Held-Karp algorithm, as a requirement for my discrete math's project. 

Before starting, you can edit the deep space objects data present in the .csv file in /deep_space_objects/dso.csv
That file is the input data for the DSOs later. If you want to edit it, follow each columns format

Afterwards, you can run the program, if everything is correct the screen should be black. To generate the complete graph, 
which is a requirement to find the shortest Hamiltonian path, press the "F" Key on your keyboard.

To then generate the shortest Hamiltonian path, press the "H" key, and wait for a couple seconds.

In the output terminal, you can see a few values printed out, such as the shortest path, the average of 1000 random Hamiltonian path as comparison, with
the best cost, worst cost, and average cost.
